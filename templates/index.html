The Parking Car Flask API Specifications

Action URL Description
Help http://127.0.0.1:7000/ Get this help page
List with parked cars http://127.0.0.1:7000/cars Lists all the cars which are currently parked
Add a car http://127.0.0.1:7000/add?car_number=CJ45WAY&tariff=hourly Example, where : car_number = CJ45WAY ; tariff = hourly or daily. Please do not use for car_number "&" as there is no URL validator implented
Remove a car http://127.0.0.1:7000/remove/id where id is the parking id place allocated. You can find a specific id by listing cars
Get available places http://127.0.0.1:7000/available_places How many available places are currently available
